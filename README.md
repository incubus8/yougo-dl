# yougo-dl


## Description
Youtube video downloader

## Usage
```bash
$ bin/yougo-dl get [YOUTUBE_VIDEO_ID] [DEST_FILE.mp4]
```

## Install

To install, use `go get`:

```bash
$ go get -d github.com/incubus8/yougo-dl
```
